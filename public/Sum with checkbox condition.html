You said:
in apple numbers how do i leave a row out of a sum if a checkbox in that row is checked?
ChatGPT said:

Yep — you can do this in Apple Numbers using the IF function along with your checkbox column. Basically, you make your sum conditional on whether the checkbox is checked.

Here’s how:

Example setup

  • Column A: Item name

  • Column B: Value (numbers to sum)

  • Column C: Checkbox (✅ = exclude from sum)

Formula for total (somewhere else, e.g. in B10):

numbers
=SUM(IF(C2:C9, 0, B2:B9))

How it works

  • C2:C9 is the range of checkboxes.

  • If a checkbox is checked (TRUE), the formula replaces that row’s value with 0.

  • If it’s unchecked (FALSE), it includes the number.

  • SUM(...) then adds up only the un-checked rows.

🔹 If you want the opposite behavior (sum only the checked rows), just flip the logic:

numbers
=SUM(IF(C2:C9, B2:B9, 0))
